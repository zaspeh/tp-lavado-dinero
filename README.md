# Sistema Distribuido: Deteccion de Patrones de Lavado de Dinero

## Tareas Recurrentes
- Comando para generar compose, levantar y ejecutar sistema: `make up`
- Comando para dar bajar y eliminar contenedores: `make down`
- Comando para ver logs `make logs`; agregar grep para filtar vista, por ejemplo: `make logs | grep -v rabbit`
-Comando para comparar salida con esperado: `make test`
